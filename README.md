# CORN AX Mentoring Service

Python Streamlit과 Google Gemini API를 사용한 스타트업 멘토링 서비스입니다.

## 기능
- **시장조사**: 경쟁사 분석 및 고객 페르소나 정의
- **문제정의**: 11단계 문제 정의 프레임워크
- **해결책**: Solution Discovery 6 method

## 로컬 실행 방법
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 사용 방법
1. Google API Key 입력
2. 사용할 Gemini 모델 선택
3. 원하는 멘토링 단계 선택
4. AI 멘토와 대화 시작
